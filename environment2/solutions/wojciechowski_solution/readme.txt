Rozwiązanie oparte o perceptron.
Parametry sieci opisane są w pliku solution-params.txt, który zawiera:
1. linia: ilość kanałów obrazu testowanego,
2. linia: maksyalna ilość iteracji podczas nauki sieci
W procesie treningu parametry zapisywane są do plików:
* neural-file-ls
* neural-file-sm
Aby uruchomić trening należy wywołać program z dodatkowym argumentem "train" (1 argument - katalog z danymi, 2 argument - typ analizy)
